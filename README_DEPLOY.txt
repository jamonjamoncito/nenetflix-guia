DESPLIEGUE GRATIS (Cloudflare Pages, recomendado):
1) Crea una cuenta en Cloudflare.
2) Ve a: Workers & Pages → Pages → Create a project.
3) Conecta tu GitHub.
4) Sube estos archivos a un repo (ej: nenetflix-guia).
5) En Cloudflare Pages:
   - Framework preset: None
   - Build command: (vacío)
   - Output directory: /
6) Deploy → tendrás una URL .pages.dev

DOMINIO guia.nenetflix.uk:
- En tu proyecto Pages: Custom domains → Add "guia.nenetflix.uk"
- DNS (si tu dominio está en Cloudflare):
  CNAME: guia  →  <tu-proyecto>.pages.dev

Después ya puedes mandar por WhatsApp:
https://guia.nenetflix.uk
